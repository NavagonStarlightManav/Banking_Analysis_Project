# Banking Dashboard – End-to-End Data Analysis with Power BI

## Project Overview

This project demonstrates an end-to-end banking data analysis workflow using Power BI. It covers the complete analytics pipeline, including data cleaning, transformation, exploratory data analysis (EDA), and the development of an interactive multi-page dashboard for business insights.

The dashboard enables stakeholders to analyze customer deposits, loans, account balances, and demographic trends through dynamic visualizations and interactive reports.



## Data Cleaning & Preparation

The dataset was cleaned and transformed to improve consistency, readability, and analytical usability.

### Data Cleaning Tasks

- Categorized customer income into three bands:
  - Low
  - Mid
  - High
- Standardized categorical variables such as Gender and Nationality.
- Created Income Bands using conditional columns in Power BI.
- Replaced branch codes (1, 2, etc.) with descriptive branch names.
- Converted gender codes into readable values:
  - `1 → Male`
  - `2 → Female`



## Exploratory Data Analysis (EDA)

### Categorical Analysis

- Gender
- Nationality

### Numerical Analysis

- Credit Card Balance
- Bank Loans
- Bank Deposits
- Checking Account Balance
- Saving Account Balance
- Estimated Income
- Superannuation Savings



## Key Insights

- A strong positive correlation was observed between:
  - Bank Deposits
  - Checking Account Balance
  - Saving Account Balance
  - Foreign Currency Account

- Customers with higher balances in one account type generally tend to maintain higher balances across other banking products.
